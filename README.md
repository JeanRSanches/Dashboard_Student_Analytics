# Dashboard_Student_Analytics
Dashboard interativo para análise de desempenho acadêmico dos alunos, com foco na identificação de fatores que influenciam o rendimento
O painel apresenta quatro indicadores principais: Desempenho Médio, Média de Horas de Estudo, Média de Horas de Sono e Percentual de alunos que participam de atividades extracurriculares
O projeto também inclui um gráfico de colunas que compara o impacto das atividades extracurriculares no desempenho, um gráfico de rosca para classificação de risco acadêmico (alto, médio e baixo) com base no índice de desempenho, e um gráfico de barras que demonstra a correlação entre diferentes fatores e o Índice de Desempenho, destacando quais variáveis mais influenciam os resultados dos alunos.
